# Asset Files

This directory should contain:

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px or larger)

For development, Expo will use default assets if these are not provided.

To add custom assets:
1. Create images with the specified dimensions
2. Place them in this directory
3. Ensure they are referenced correctly in `app.config.ts`

## Recommended Tools

- [Figma](https://www.figma.com/) - Design icons and splash screens
- [App Icon Generator](https://www.appicon.co/) - Generate all required icon sizes
- [Expo Asset Tools](https://docs.expo.dev/guides/app-icons/) - Official Expo documentation
